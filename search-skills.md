# Searcg Skills in Cybersecurity
## Overview
This room focused on developing research skills for cybersecurity professionals.
## Key Concepts Learned
### Evaluation of Search Results
- Assessing credibility of sources.
- Checking publication dates.
- Cross-referencing information.
- Identifying bias or unreliable content.
### Search Engines
- Using general search engines effectively.
- Refining searches to get accurate technical results.
### Specialised Search Engines
- Using security-focused platforms to find vulnerabilities and exploits.
- Researching CVEs and exploit databases.
### Technical Documentation
- Understanding the importance of reading official documentation.
- Learning how to extract relevant technical details.
### Social Media in Cybersecurity
- Monitoring security communities for threat intelligence.
- Staying updated on new vulnerabilities and attack trends.
## Reflection
Strong research and search skills are essential in cybersecurity, as professionals must constantly investigate threats, vulnerabilities, and solutions.
