# Finding Hidden Pages - Enumeration
## Overview
This room focused on discovering hidden directories on a website.
## What I Learned
-Websites may contain hidden directories such as /admin or /backup.
-Attackers use tools like dirb to discover hidden pages.
-Exposed admin or backup pages create security risks.
## Tools Used
-dirb
## Risk Perspective
Exposed hidden directories indicate weak access controls and poor configuration management.
