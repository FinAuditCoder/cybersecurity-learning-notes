# Defensive Security Introduction
## Overview
This room introduced defensive security, which focuses on protecting systems from cyber attacks.
## What I Learned
- Defensive security involves monitoring, detecting, and responding to threats.
- Security Operations Centre (SOCs) monitor systems and investigate suspicious activity.
- Tools like firewalls and intrusion detection systems help protect networks.
## Offensive vs Defensive
- Offensive security tests systems by attacking them ethically.
- Defensive security protects systems and responds to attacks.
## Risk Perspective
- From a risk perspective, defensive security ensures that threats are detected early and incidents are managed to reduce impact.
